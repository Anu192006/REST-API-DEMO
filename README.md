# REST API Demo

This project demonstrates how REST APIs work and how to integrate a public API using JavaScript.

## Features
- Fetches data from a public REST API
- Displays dynamic data on a web page
- Uses HTML and JavaScript

## Technologies Used
- HTML
- JavaScript
- REST API

## How It Works
The application sends an HTTP request to a public API using JavaScript (fetch).
The response is processed and displayed dynamically on the webpage.
